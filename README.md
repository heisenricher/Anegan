Aleph is an elegant, minimal, and completely offline application that handles conversions for **Documents, Images, Video, and Audio**. Designed with privacy and simplicity in mind, it supports deep advanced settings (like exact file size targeting) without sacrificing its clean, Apple-like user experience.

*(Once downloaded, tap the APK file on your Android device to install. Make sure "Install Unknown Apps" is enabled in your settings.)*

---

## ✨ Features
- **Offline First**: No tracking, no ads, no cloud uploads. Everything happens on your device.
- **Documents**: Format shifting and PDF tools.
- **Images**: Exact size targeting, precise compression, and format conversion (JPG, PNG, WEBP).
- **Media Engine**: Powerful video downscaling, format switching, and fast audio extraction powered by FFmpeg.
- **Secure History**: Review past conversions via Biometric unlock (Fingerprint/Face).

## 🛠️ For Developers
The application is built entirely using modern Android standards:
- **Jetpack Compose** for a butter-smooth 120Hz UI
- **Clean Architecture** (Multi-module)
- **Room Database** for local history
- **Biometric API** integration

To build from source, clone the repository and open it in Android Studio.
